# Mateoaventura 🐾🦸‍♂️

¡Ayuda a **Mateo** a rescatar a los **animales perdidos** en **Las Jaras** (Córdoba) de unos **aliens secuestradores**! Usa tu **tirachinas** para defender a los animales y llevarlos al **punto de rescate**.

## 🎮 Cómo jugar

Abre `index.html` en cualquier navegador moderno.

### Controles

| Tecla | Acción |
|-------|--------|
| `← →` o `A / D` | Mover a Mateo |
| `Espacio` / `↑` / `W` | Saltar |
| `J` / `K` | Disparar con la tirachinas 🪨 |
| `Esc` / `P` | Pausa |

En **móvil**: botones táctiles en pantalla (mover, saltar, disparar).

### Objetivo

- 🐾 **Rescata** a todos los animales perdidos de cada nivel
- 👽 **Derrota** a los aliens con la tirachinas (o saltando encima)
- 🏠 **Llega** al punto de rescate con todos los animales
- ⏱️ **Supera** los 3 niveles en el menor tiempo posible

## 🗺️ Niveles

1. **El Campo** — Campos abiertos de Las Jaras con vallas y muretes
2. **El Arroyo** — Barrancos, puentes de madera y agua
3. **La Dehesa** — Atardecer dorado entre encinas, máxima dificultad

## ✨ Características

- 100% HTML5 Canvas + JavaScript (sin dependencias)
- Sonido sintetizado en tiempo real (Web Audio API)
- Compatible con móvil y escritorio
- Récord guardado en localStorage
